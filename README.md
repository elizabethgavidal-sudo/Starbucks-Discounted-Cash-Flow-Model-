# Starbucks-Discounted-Cash-Flow-Model-
Developed a discounted cash flow (DCF) valuation model for Starbucks using public financial statements and SEC Form 10-K filings to create revenue forecasts, WACC, terminal value, and sensitivity analysis.

Tools Used: Microsoft Excel 

Key Features: Excel-Based DCF Model with Sensitivity Analysis, DCF Valuation, and Financial Forecast

This is a **mock portfolio project** created to demonstrate Microsoft Excel and financial modeling using publicly available financial data from Starbucks' Annual Reports.
